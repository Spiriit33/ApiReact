<h3>Documentation API</h3>

<h4>users/login</h4>
<ul>
        <li>POST</li>
        <li>2 paramètres : email de type <b>string</b>, password de type string</li>
        <li>Retourne une réponse JSON de status 200 contenant l'api token de l'utilisateur</li>
</ul>
